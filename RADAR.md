## How to Add a Project

Register a repo by adding it here and ensuring it has the canonical state files:
- `AGENT.md`
- `PROJECT_STATUS.md`
- `NEXT_STEPS.md`
- `DECISIONS.md`
- `session-log/`

Use the templates in `templates/` to bootstrap a new project.
